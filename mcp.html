Model Context Protocol

The Model Context Protocol (MCP) is the underlying architecture that powers the interactive elements within DesignHub.

It acts as a standardized interface between the visual layer (your animations) and the state management system, ensuring that every transition is smooth, performant, and context-aware.

Core Principles

  • State Preservation: Animations remember their state even when unmounted.
  • Context Injection: Components adapt to their container size and theme automatically.
  • Event Propagation: Interactive elements broadcast events efficiently without bubbling chaos.

Integration

You don't need to manually configure MCP in your projects. It's built into the core `designhub-core` package, specifically designed to be invisible while doing the heavy lifting.
